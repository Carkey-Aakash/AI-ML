# Statistics Coding Exercises

This repository contains Python coding exercises and implementations for learning basic and intermediate statistics concepts. All computations are implemented manually without relying on libraries like `pandas` or `numpy`.

## Topics Covered

### Day 1 – Basics of Statistics
- Population vs Sample
- Range, Class Intervals
- Mean, Variance, Standard Deviation  

**Coding Tasks:**
- `mean()`
- `variance()`
- `std()`
- Frequency table

---

### Day 2 – Central Tendency
- Mean, Median, Mode
- Weighted Mean  

**Coding Tasks:**
- `mean()`
- `median()`
- `mode()`
- `weighted_mean()`

---

### Day 3 – Dispersion
- Range, Interquartile Range (IQR)
- Variance, Standard Deviation
- Coefficient of Variation  

**Coding Tasks:**
- `percentile()`
- `IQR()`
- `coefficient_of_variation()`

---

### Day 4 – Probability Foundations
- Conditional Probability
- Bayes Theorem  

**Coding Tasks:**
- Monte Carlo simulations
- Bayes theorem function

---

### Day 5 – Distributions
- Normal, Binomial, Poisson
- Z-scores  

**Coding Tasks:**
- `normal_pdf()`
- `binomial()`
- `poisson()`
- `z_score()`
